# Aéromovie - Projet NSI 2023

Site éducatif sur l'aéronautique et le cinéma avec quiz interactif.

## Auteurs
- Timothé Luzzatto (HTML/JS)
- Tom Pitman (CSS/JS)

## Fonctionnalités
- Popup d'accueil
- Changement de thème
- Quiz interactif
- Diaporama d'images
- Glossaire
- Compteur de visites

## Navigation
Ouvrir `index.html` dans Chrome/Firefox.
